# dotfiles

My personal dotfiles.

# Usage

```
$ git clone https://github.com/kevingo/dotfile.git && cd dotfile && source bootstrap.sh
```

# Installation

- Execute the `brew.sh` to install Homebrew and necessary softwares.

```
$ ./brew.sh
```

- Execute the `nvm.sh` to install nvm.

```
$ ./nvm.sh
```
